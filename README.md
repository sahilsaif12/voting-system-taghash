    Frontend- React js
    Backend - Node js, Express js

## Set up the application
1. clone the repository-
   
   `
   git clone git@github.com:sahilsaif12/voting-system-taghash.git
   `
2.   ` cd client`
3. Install the dependancies and run the app
   
     `
       npm install
   `
   <br/>
   `
     npm run dev
   `
   
4. For backend-
   
   ` cd client`
    <br/>
    `npm install`
   <br/>
   `
     npm run dev
   `
